Detailed Description

The action converts graphs into a matrix based on a cost-map, the first argument can be a string with dense or sparse for defining the returned matrix, after that a cost-map cann be set or a fixed numerical value for defining the cost of an edge, all other arguments are graphs, for each graph the matrix and vertex list is returned, the action never fails

[M1|V1|M2|V2] = graph/adjacencymatrix( "dense|sparse", CostMap, Graph1, Graph2 );
[M3|V3|M4|V4] = graph/adjacencymatrix( CostMap, Graph1, Graph2 );
[M1|V1|M2|V2] = graph/adjacencymatrix( "dense|sparse", 1, Graph1, Graph2 );
[M3|V3|M4|V4] = graph/adjacencymatrix( 1, Graph1, Graph2 );
Note
the cost-map does not need an entry for each edge non-existing edges have got on default zero costs with 1
